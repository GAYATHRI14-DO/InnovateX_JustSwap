# **App Name**: SwapIt

## Core Features:

- User Registration & Profile: Allow users to create accounts, manage their profiles, and view their listed items and swap history.
- Item Listing: Enable users to post items they wish to swap, including details like description, condition, category, and images.
- AI-powered Item Description Helper: Provide a tool that uses generative AI to suggest or refine item descriptions based on item title, category, and a few keywords provided by the user.
- Browse and Search Items: Users can browse all available items, filter by category, condition, location, and search by keywords.
- Propose Swaps: Allow users to send swap proposals for items they are interested in, offering one or more of their own listed items in exchange.
- Manage Swap Proposals: Enable users to view incoming swap proposals, accept or decline them, and see the status of their outgoing proposals.

## Style Guidelines:

- Primary color: A rich, energetic green for key UI elements, embodying sustainability and growth (#1F933C).
- Background color: A very light, subtle tint of green, ensuring readability and a fresh, clean feel (#EDF5EF).
- Accent color: A vibrant, lime-green/yellow for call-to-action buttons and interactive highlights, creating strong contrast and dynamism (#C1F53D).
- Headlines and prominent text: 'Space Grotesk' (sans-serif) for a modern, slightly techy, and bold aesthetic.
- Body text and detailed content: 'Inter' (sans-serif) for optimal legibility across various screen sizes and text lengths.
- Employ a consistent set of clean, modern, and friendly line-art or subtly filled icons that complement the platform's focus on community and exchange.
- Feature a hero section similar to the provided image, with a prominent headline and clear call to action. Use a clean, card-based grid layout for displaying item listings.
- Incorporate subtle animations for interactive elements, such as item additions, proposal sending/receiving, and button hovers, to enhance user engagement and provide feedback.
